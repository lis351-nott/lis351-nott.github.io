COMP SCI 300/400 - Programming

In this course, you got to learning the basics of programming through Java. The basics here include, but not limited to,

  • primitive data types and their wrappers
  • object oriented programming (i.e. objects, classes, inheritance, etc)
  • if-else, for loop, switch
  • Some basic data structures (i.e. linked lists, binary search trees, skip list, etc)

If you already have some programming experience before, these two class are very good reviews for the core fundamental concepts of programming

However, with that said, if you are already familiar with these concepts, these classes are going to be very boring. You might be given some group projects to work on, but they are still far apart from what people really use on real projects nowadays.
