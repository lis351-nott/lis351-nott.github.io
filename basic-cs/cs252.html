COMP SCI 252 - Introduction to Computer Engineering

I would not be surprise if you say you have used computers for a really long time, maybe since your childhood.

With that said, this course's main question is "Do you want to know how the first computer was built?"

No no, I did not only mean just "how you piece transisters together to create logic gates, which are miniscule but inherent parts of all computers" or "how assembly code works on x86, one of the first complex instruction set computer (CISC)?"

I also mean "the history of computer". Like, who was the first person to actually create something that resembles a computer? What was defined to be a "computer" in the past?

This class might look boring because you won't know how all of these are related to understanding how "modern computers" work until MUCH later in the curriculum.

However, I guarantee you that this class is really important to build your mental image on how computer, the machine you have been using for who knows how long, actually functions.

If you slack off on this course, you are gonna feel it in COMP SCI 354, the sequel of this one.
